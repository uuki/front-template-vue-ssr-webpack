# front-template-vue-ssr-webpack

---

## Enviroment

- Node.js >= 6.11.0
- npm >= 3.10.10

### Global npm Module

- nodemon
- concurrently

## Setup

```bash
# ./
$ npm install [or yarn install]
$ npm run build [or yarn run build]
```

### Starting Local API Gateway ( recommend tmux )

```bash
$ npm run serve
```

### Starting Build Server ( recommend tmux )

```bash
$ npm run watch
```
